# Cost Optimization Framework (Simulated)

Category: Cost Optimization

- Idle compute (<10% usage): highest immediate savings opportunity.
- Oversized compute (<40% usage with high spend): right-size.
- Orphaned storage/unattached assets: decommission quickly.
- Commitment models (reserved/committed use): reduce steady-state cost.

Methodology note:
15-30% savings is a modeled potential range for simulated workloads.
